# Test-Automation-Framework-for-SOVD-UDS-Diagnostics
Python-based ECU diagnostic test automation using UDS (ISO 14229). Covers session control, security access seed-key, DID read/write, and DTC fault memory — 20 automated test cases with a simulated ECU over TCP. Intentionally built as a simulation hence no hardware required. Maps directly to real CAN/ DoIP setups via python-can and udsoncan.
